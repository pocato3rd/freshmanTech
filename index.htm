Our Requirments and What I Did

    This website was programmed using eMacs on the Terminal. It was created for my Freshman Technology class at the Batle Creek Area Math and Science Center. We are to create a website that portrays what we've learned in class. The three main languages we learned to make a website like the one you see before you are Hyper Text Markup Language 5, Cascading Stylesheets, and Javascript. We put our knowledge of these three languages together to create a website that works and flows properly.

    We were required to create 4 different pages:

  • a home page, which you see before you,
  • a summary of the BCAMSC,
  • a summary of our home schools, mine being Pennfield,
  • a working Javascript page, and
  • I added another Javascript page as well
  • I had time and was bored, so I added yet another Javascript page

    In addition to these HTML pages, we were required to make a stylesheet that links to the pages. I actually created 8 different stylesheets that link throughout my website.

  • techsite.css - This is the overarcing stylesheet
    • index.css - The stylesheet linked to my home page
    • bcamsc.css - The stylesheet linked to my BCAMSC summary
    • penn.css - The stylesheet linked to my Pennfield Summary
    • jscript.css - The stylesheet linked to my Javascript page.
    • carousel.css - This stylesheet is also linked to my Javascript page, it allows my code screenshots to scroll.
    • draw.css - This stylesheet links to my second Javacript page
    • dropdown.css - This stylesheet is also linked to my second Javascript. It allows the dropdown menus to work
    • space.css - This stylesheet is linked to my third Javascript page.

    One of the requirements for our website was an ease of naviagtion between the different pages. Knowing this, I created the slide out menu at the top left. This menu uses jQuery and Javascript to slide out on click. Using if statements, the jQuery understands how to change the body and position of the bar. The code for this simple change took me a little over an hour to debug and correct.

    We were given minimum requirements for the different HTML elements we add to our page. This includes, but wasn't limited to: lists, tables, GIFs, button, and font sizes.

My Difficulties

I ran many tests to make sure it worked and it being more 600+ lines of code, things didn't work so well for a long time. Most of my errors were missing a semi-colon or a simple misspelling. However, when things did work, I became estatic. I ended up overcoming each challenge and I am very proud of how it turned out.

    Also on my first Javascript page, I had tried to make the screenshots of my code fade in and out like some online carousels. I got the first picture to fade into the next, but after that I couldn't get any further. I even tried to look at some other carousel code, but yet again, I couldn't decipher how it should work. After much frustration, I decided I would change it to what it is now, a box that has a set height that the pictures reside in.

    On my second Javascript page I had a quite a bit of trouble doing mainly two things: Having the page understand when the mouse is being held and dragged and making the dropdowns. The page greatly utilizes these two events and it took me awhile to figure out.

    Another difficulty I had was actually writing the text for the page you're on now, my Home page. At first I thought I'd make it about myself, but I'm not very good at talking about myself. I actually wasted an entire day thinking of what to write. However, I finally decided to write the different requirements we had and how I achieved those requirements.
